Challenge en Java - Juego de Carros

En este juego se te permitirá elegir una cantidad de personajes para
competir con tus amigos en la cantidad que desees de recorridos, se 
trata de un juego de azar donde el movimiento de los vehiculos estará
basado en un dado, al finalizar el juego se entregarán las estadisticas
de los recorridos y el general.
